    데이터저장소
    신청

    데이터뱅크 내 저장하고자 하는
    질병의 데이터저장소 신청

    [학습 데이터 관리 >
    질병 데이터 저장 신청] 메뉴를 통해 신청

  • 2

    생성 알림
    메일 전달

    데학습 포탈 관리자가 저장소신청 정보를
    기반으로 클라우드의 데이터뱅크 내
    데이터저장소 생성한 후 신청 병원에
    알림 메일 전달

  • 3

    데이터뱅크에
    업로드

    공통 플랫폼의 대량데이터업로더를
    통한 데이터를 PC > 클라우드 데이터뱅크의
    병원 질병의 데이터저장소 업로드

  • 4

    수집 현황
    모니터링

    데이터 수집 현황을 통해
    수집된 데이터량 모니터링

    데이터
    공개 신청

    데이터뱅크 또는 클라우드 내

    [학습 데이터 관리 >
    질환 데이터 공개 신청] 메뉴를 통해 신청

  • 2

    학습데이터
    등록

    학습 포탈관리자가 학습 데이터
    등록 가능여부 체크 후
    학습 데이터 등록이 진행되며,
    등록된 화면은 전체 사용자가
    조회 가능

  • 3

    학습데이터
    사용 신청

    질환별 데이터를 조회 후
    학습 데이터 사용 신청

  • 4

    승인 여부
    선택

    학습데이터 사용 신청한
    기업정보를 확인 한 후 기업의
    신청에 대해 승인/거절

    [학습 데이터 관리 >
    학습 데이터 사용 승인] 또는
    [마이페이지 > 기업 요청 데이터 보기]
    메뉴를 통해 승인/거절

  • 5

    질환데이터
    전달

    승인된 학습 데이터 신청 정보를
    기반으로 기업에게 독립된
    학습 데이터 저장소 공간을
    생성하여 질환 데이터를 전달

    승인 여부
    확인

    [학습 데이터 관리 > 학습 데이터 사용 신청] 또는
    [마이페이지 > 데이터 사용 신청] 메뉴를 통해

    학습공간에서
    데이터 활용

    학습공간 신청 또는 클라우드 내 학습서버를
    구성하여 학습데이터저장소 연결하여
    데이터를 활용하여 학습함

    [학습 데이터 관리 > 학습 데이터 사용 신청] 또는
    [마이페이지 > 데이터 사용 신청] 메뉴를
    통해 연결 요청
